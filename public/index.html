Lyrics API Documentation

Explore the functionality of the Lyrics API

Usage

To get lyrics for a specific track by ID, make a GET request to: /getLyrics/:trackId

To get lyrics by musician and track name, make a GET request to: /getLyricsByName/:musician/:track

Donation

If you find this service helpful, consider making a donation to support its maintenance and development.

Example

Get lyrics by track ID:
/getLyrics/123456

Get lyrics by musician and track name:
/getLyricsByName/artistName/trackName

For more details, refer to the API documentation. For issues, contact support@example.com
